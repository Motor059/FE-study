C++ 소개

언어 소개

C++은 1979년에 비야네 스트롭스트룹(Bjarne Stroustrup)이 개발한 프로그래밍 언어입니다.
C++은 C 언어의 상위 호환 언어로, C 언어 의 모든 기능을 포함하면서도 객체 지향 프로그래밍, 템플릿, 예외 처리 등 다양한 기능을 제공합니다.
C++은 강력한 기능과 유연성을 제공하는 프로그래밍 언어로, 시스템 프로그래밍, 게임 개발, 그래픽스, 임베디드 시스템 등 다양한 분야에서 널리 사용되고 있습니다.

핵심 개념

  • 변수 선언 : int, float, double, char 등 다양한 데이터 타입으로 변수를 선언 가능
  • 제어문 : if, else, switch, for, while 등 다양한 제어문을 사용 가능
  • 함수 : 코드를 모듈화하고 재사용 가능
  • 클래스 : 객체 지향 프로그래밍 구현 가능
  • 포인터: 메모리의 주소값을 직접 제어하여 세밀한 프로그래밍 가능
  • 템플릿: 다양한 데이터 타입에 대해 동일한 코드를 작성 가능(타입인수화)
  • 예외 처리: try, catch, throw를 사용하여 오류 처리 가능
  • 입출력 : cin과 cout을 사용하여 입출력 가능
  • 네임스페이스: namespace로 이름 충돌을 방지 가능
  • STL(Standard Template Library): vector, string 등 강력하고 편리한 표준 라이브러리 사용 가능

다음은 "Hello, World!"를 출력하는 간단한 예제입니다. 

    #include <iostream>
    using namespace std;
    int main() {
        cout << "Hello, World!";
        return 0;
    }
