## **JS와 HTML 상호작용**
- Counter App 페이지를 html의 h1, p, button tag를 이용해서 구성하고 동작을 적용하기 위해 script tag까지 이용했습니다.
- js파일을 만들고 변수와 각 변수에 대한 id를 선언한 뒤 버튼 동작을 함수로 구현하였습니다. 
- 최종적으로 html에서 동작이 필요한 tag에 id를 입력하여 js와 상호작용할 수 있도록 연결하였습니다. 
- script 태그는 body의 맨 아래에 있는데 html 해석 속도 문제와 순서가 꼬일 수 있다는 문제 때문입니다.(자세한 것은 따로 공부)

**사용한 주요 태그 및 기능**:
    - `<h1>`: 글자의 제목(Heading)
    - `<p>`: 문단(Paragraph)
    - `<button>` : 상호작용 가능한 버튼 생성 / 
    - `<script>` : html을 js와 연결
      - ex : `<script src = "app.js"></script>`
    - `id` : id당 하나의 고유한 태그를 인식
    - `getElementById` : 특정 html 요소를 id로 선택
    - `textContent` : id에 해당하는 요소의 내용을 바꾸기 위해 사용
    - `addEventListener` : 클릭하는 등의 이벤트를 감지하여 실행

**JavaScript란?**
- HTML만으로는 웹 문서가 일방적으로 정보를 주는 정적인 형태이기 때문에 서로 소통이 가능한 동적인 형태로 만들기 위한 언어라는 것을 배웠습니다. 
- 브라우저에는 JS 코드를 해석하고 실행하는 자바스크립트 엔진이 내장되어 있는데 대표적으로 구글 크롬의 V8 엔진이 있다는 것을 배웠습니다.
- 브라우저가 아닌 환경에서도 자바스크립트를 실행할 수 있게 해주는 Node.js가 등장했다는 것을 배웠습니다.

## **2주차 과제 수행**
- +10, -10 버튼은 기존 +1, -1 구조를 그대로 가져온 뒤 count 연산 부분만 가볍게 건드려서 구현할 수 있었습니다.

**JavaScript 기본 문법**
- **주요 문법**:
    - `변수 선언`: let은 재할당이 가능한 변수, const는 재할당이 불가능한 상수를 선언할 때 사용합니다.
    - `연산`: 산술 연산자 : +, -, *, /, % | 할당 연산자 : =, +=, -= | 비교 연산자: ==, === (값과 타입이 모두 같음), !=, !==, >, <
    - `반복문`: 일반 코드 작성법과 같이 for문, while문을 사용합니다.
    - `조건문`: 일반 코드 작성법과 같이 if, else if, else를 사용합니다. 
    - `함수`: 코드를 재사용할 수 있게 해주는 역할을 수행하는 코드로 function 키워드를 이용해 선언합니다.
